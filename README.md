# PlainFly
闪烁的星星，自由飞翔的飞机，摇曳的小树（自定义+属性动画+贝塞尔曲线+VectorDrawable）
贝塞尔曲线公式：<html><body><br/></bory></html>
一阶公式：<html><body><br/></bory></html>
(1-t)*P0 + t*P1    t=[0,1]
二阶公式：
(1-t)^2*P0 + 2*(1-t)*t*P1 + t^2*P2   t=[0,1]
三阶公式：
(1-t)^3*P0 + 3*(1-t)^2*t*P1 + 3(1-t)*t^2*P2 + t^3*P3  t=[0,1]
